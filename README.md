# Página de Compras Vulnerable

Este repositorio contiene una página web de compras en línea diseñada intencionalmente para ser vulnerable, con el propósito de practicar y aprender principios de ciberseguridad.

El proyecto fue asignado por el profesor Guillermo Mendoza Hernández para su clase "Seguridad en el Desarrollo de Aplicaciones" en la Universidad Tecnológica San Juan del Río (UTSJR).

## Objetivo

- Ofrecer un entorno de aprendizaje donde se puedan identificar y analizar vulnerabilidades comunes en aplicaciones web.
- Practicar técnicas de detección, análisis y mitigación de riesgos en un sitio de comercio electrónico simulado.

## Uso

- No se trata de una aplicación destinada a producción.
- Está pensada para ejercicios académicos y prácticas de seguridad.

## Curso

- Asignatura: Seguridad en el Desarrollo de Aplicaciones
- Profesor: Guillermo Mendoza Hernández
- Institución: Universidad Tecnológica San Juan del Río (UTSJR)

# Correr la página
